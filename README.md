Overview



My Social App is a simple social media web application built using Node.js and Express.js.

Users can create posts, view feed, and interact with basic social media features through REST APIs.



🚀 Features

Create new posts

View all posts in feed

Like posts (if implemented)

Simple REST API backend

Lightweight database (SQLite / JSON based storage)

🛠️ Technologies Used

Node.js

Express.js

HTML

CSS

JavaScript

SQLite

📂 Project Structure

server.js - Main backend server

package.json - Project dependencies

node\_modules/ - Installed packages

social.db - Database file

frontend/ - UI files (if available)

⚙️ Installation

1\. Clone the repository

git clone https://Pugazh2123@github.com/Pugazh2123/CodeAlpha\_my-social-app.git

2\. Move into project folder

cd CodeAlpha\_my-social-app

3\. Install dependencies

npm install

4\. Run the server

node server.js

5\. Open in browser

http://127.0.0.1:3000

📡 API Endpoints

GET /api/posts → Get all posts

POST /api/create-post → Create new post

POST /api/like-post → Like a post
